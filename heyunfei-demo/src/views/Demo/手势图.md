# 属性

 参数       | 说明  | 类型  | 是否必须 | 可选值  | 默认值  |
-------- | --------------- | ------ | ---- | ---- | ---- |
point-default | 默认的手势（初始手势） | String | 否 | 九位1-9的数字（不可重复）  | --  |
point-correct | 被匹配的手势（正确手势） | String | 否 | 九位1-9的数字（不可重复）  | -- |



# 方法

无

# 事件

 事件名称| 说明    | 回调参数 |
 ------------- | ----- | ------------ |
 draw-end | 画完手势后 | val（画的手势） |
 success | 验证成功 | val（画的手势） | 
 cancel | 验证成功 | val（画的手势） |
